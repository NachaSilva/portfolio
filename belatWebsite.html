BELAT NEW WEBSITE

UX UI Design | Web Developer | Support

In my role as a UX/UI Designer and Web Developer, I led the restructuring of Belat’s corporate websites, an investment bank, optimizing both the global site and creating localized versions for the countries where the company operates. The main goal was to enable each regional office to create tailored content aligned with their local offerings, while also improving key metrics such as country-specific objectives and targeted campaigns.

From a UX perspective, I conducted extensive research with both global and local users, including internal clients. The process involved creating interactive prototypes, designing the information architecture, and implementing user flows to ensure an intuitive and efficient user experience. I also collaborated on the development of brand-aligned graphic guidelines to maintain a unified visual identity across all markets.

Regarding web development, after gathering the necessary feedback and validating new landing page prototypes or improvements, I was responsible for the responsive development, ensuring that each page adapted seamlessly across various devices and browsers. Additionally, I optimized the overall performance of the sites, focusing on faster load times and improved SEO results.

Platform:Belat Website, Brasil Website, Argentina/Uruguay Website, Events Website

Methodology: Agile

Duration: 1 year

UX/UI Tools: Figma, Adobe

Server Digital Ocean

CMS WordPress

Visual Builder Elementor

Frontend: HTML, Javascript, CSS, jotform

Backend: PHP

Future: Implement the new site map to unify the sites to improve the customer journey. Separate Chilean content with the global site.
